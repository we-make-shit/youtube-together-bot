# Installing
- clone this repo
- rename `.env.example` to `.env`
- add your token in `.env` :: `DISCORD_TOKEN=YOUR_BOT_TOKEN`
- now run the bot with `npm start`

# YouTube Together
- run `a!yttogether <channelid>`
- join with the invite created by your bot

# Default command prefix
`a!`
